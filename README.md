### Dependencies

You will need to run this app:
- `yarn` (https://yarnpkg.com/)

### Installing

```
yarn add
```

### Compiles and Running

```
yarn run serve
```

### Compiles and build for production

```
yarn run build
```

### Run tests
```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```

### Run your unit tests

```
yarn run test:unit
```
